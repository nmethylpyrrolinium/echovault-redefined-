# EchoVault Visual Identity

EchoVault should feel like a private emotional observatory: intimate, luminous, calm, and premium. Use cinematic detail to make memories feel treasured, not gamified.

## Visual language

- Dark, soft backgrounds with subtle depth gradients.
- Luminous orbs as emotional anchors, not generic blobs.
- Thin constellation geometry to imply memory relationships.
- Particles as atmosphere, never confetti.
- Glow as soft emotional resonance; avoid neon overload.

## Restraint rules

- One dominant glow color per moment, with one supporting accent at most.
- No more than three animated layers in a hero surface: e.g. orb drift, particles, constellation lines.
- Avoid rapid looping, bouncing, elastic overshoot, and noisy randomness.
- Prefer slow parallax, breathing scale, opacity drift, and reveal choreography.
